Os Meus Questionários

No painel Os Meus Questinários tem acesso aos questionários criados por si ou para os quais lhe deram permissão de Administrador.
Em qualquer questionário tem acesso às informações de Número de Recolhas, Estado e também pode Explorar Resultados onde lhe são apresentadas informações gerais sobre os participantes.


Nos questionários com Recolhas Abertas é possível:
  • Recolha Presencial: Efetuar uma recolha com um participante presencial.
  • Recolha Remota: Gerar link para uma participação remota e copiá-lo ou enviá-lo por e-mail.
  • Configurar Recolhas: Associar um questionário de treino e/ou uma explicação das escalas Self-Assessment Manikin ao questionário.
  • Fechar Recolhas: Proibir recolhas para o questionário.
  • Clonar: Clonar o questionário.
  • Editar: Editar o questionario se o mesmo ainda não tiver recolhas.
  • Descarregar Recolhas: Descarregar um ficheiro de formato .xlsx com todos os dados recolhidos dos participantes.
  • Exportar Questionário: Descarregar um ficheiro de formato .psyconfig com as definições do questionário.
  • Permissões Acesso: Verificar permissões de acesso ao questionário e convidar utilizadores para acesso ao mesmo.

Nos questionários com Recolhas Fechadas é possível:
  • Abrir Recolhas: Permitir recolhas para o questionário.
  • Configurar Recolhas: Associar um questionário de treino e/ou uma explicação das escalas Self-Assessment Manikin ao questionário.
  • Clonar: Clonar o questionário.
  • Editar: Editar o questionario se o mesmo ainda não tiver recolhas.
  • Descarregar Recolhas: Descarregar um ficheiro de formato .xlsx com todos os dados recolhidos dos participantes.
  • Exportar Questionário: Descarregar um ficheiro de formato .psyconfig com as definições do questionário.
  • Permissões Acesso: Verificar permissões de acesso ao questionário e convidar utilizadores para acesso ao mesmo.
  • Arquivar: Arquivar o questionário. É impossível voltar a utilizá-lo após esta opção.

Nos questionários Arquivados é possível:
  • Clonar: Clonar o questionário.
  • Descarregar Recolhas: Descarregar um ficheiro de formato .xlsx com todos os dados recolhidos dos participantes.
  • Exportar Questionário: Descarregar um ficheiro de formato .psyconfig com as definições do questionário.
  • Apagar: Apagar definitivamente o questionário.

Questionários Partilhados Comigo

No painel Questionários Partilhados Comigo tem acesso aos questionários partilhados consigo e para os quais não tem permissões de Administrador.
Em qualquer questionário tem acesso às informações de Número de Recolhas, Estado e também pode Explorar Resultados onde lhe são apresentadas informações gerais sobre os participantes.


Nos questionários com Recolhas Abertas é possível:
  • Recolha Presencial: Efetuar uma recolha com um participante presencial.
  • Recolha Remota: Gerar link para uma participação remota e copiá-lo ou enviá-lo por e-mail.
  • Descarregar Recolhas: Descarregar um ficheiro de formato .xlsx com todos os dados recolhidos dos participantes.

Nos questionários com Recolhas Fechadas é possível:
  • Descarregar Recolhas: Descarregar um ficheiro de formato .xlsx com todos os dados recolhidos dos participantes.

Nos questionários Arquivados é possível:
  • Descarregar Recolhas: Descarregar um ficheiro de formato .xlsx com todos os dados recolhidos dos participantes.

Criar ou Importar Questionários


Para criar um questionário:
  • Clique em Criar Questionário no canto inferior direito.

Para importar um questionário:
  • Clique em Importar Questionário no canto inferior direito.
  • Pesquise o questionário que pretende importar e de seguida clique no botão Carregar.
